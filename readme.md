# Chebyshev Linkage Simulation

You can change the length of each linkage and simulate what happens as a result.

![Chebyshev Linkage Simulation](image/image.png)

## Requirement

* python3
* numpy

## Usage

```bash
python3 chebyshev_linkage_simulation.py
```
